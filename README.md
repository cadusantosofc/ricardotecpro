# Exercícios Java

Exercícios básicos para treinar programação em Java.

## Como usar

- Abra o projeto na IDE.
- Os arquivos dos exercícios estão na pasta `src`.
- Execute o que quiser testar.

## Exercícios

Terreno, Retângulo, Idades, Soma, Troco, Círculo, Pagamento, Consumo, Medidas, Duração.
